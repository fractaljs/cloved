# Usage Guidelines

Checkboxes allow users to select one or multiple options from a list. They can be used to:
- Enable/disable features or settings
- Select multiple items from a list
- Accept terms and conditions
- Filter or sort content

## Best Practices
- Use clear, concise labels
- Group related checkboxes logically
- Provide clear feedback for selected state
- Use appropriate spacing between options
- Consider using indeterminate state for parent-child relationships
