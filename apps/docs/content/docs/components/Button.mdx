# Usage Guidelines

Buttons are interactive elements that trigger actions or events. They can be used to:
- Submit forms
- Trigger actions
- Navigate to different pages
- Confirm or cancel operations

## Best Practices
- Use clear, action-oriented labels
- Maintain consistent button hierarchy (primary, secondary, tertiary)
- Ensure sufficient contrast for accessibility
- Use appropriate button sizes based on context
- Include loading states for asynchronous actions.
