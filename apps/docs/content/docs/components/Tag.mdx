# Usage Guidelines

Tags are compact elements that represent categories, labels, or status. They can be used to:
- Display categories
- Show status indicators
- Label items
- Filter content
- Highlight attributes

## Best Practices
- Use clear, concise text
- Choose appropriate colors for different types
- Maintain consistent styling
- Consider interactive states
- Ensure sufficient contrast
- Use appropriate sizing
- Support removal when needed
