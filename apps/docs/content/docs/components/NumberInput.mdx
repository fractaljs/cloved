# Usage Guidelines

NumberInput allows users to input numerical values. They can be used to:
- Enter quantities
- Input measurements
- Set numerical parameters
- Calculate values
- Specify ranges

## Best Practices
- Set appropriate min/max values
- Include step controls when needed
- Provide clear validation
- Support keyboard input
- Consider decimal places
- Handle invalid inputs gracefully
- Use appropriate formatting
