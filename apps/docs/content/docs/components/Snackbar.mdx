# Usage Guidelines

Snackbars provide brief notifications about app processes. They can be used to:
- Show success messages
- Display error notifications
- Provide action confirmations
- Show system status updates
- Present temporary information

## Best Practices
- Keep messages concise and clear
- Use appropriate duration for visibility
- Include action buttons when needed
- Position consistently on screen
- Avoid multiple snackbars
- Consider mobile placement
- Use appropriate styling for different message types
