# Usage Guidelines

DateRangePicker allows users to select a range of dates. They can be used to:
- Filter data by date range
- Schedule events
- Set time periods
- Book appointments
- Generate reports

## Best Practices
- Provide clear date format
- Include helpful presets
- Show calendar view
- Support keyboard navigation
- Validate date ranges
- Consider timezone handling
- Ensure mobile-friendly interface
