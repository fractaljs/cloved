# Usage Guidelines

Sidebars provide navigation and contextual information alongside main content. They can be used to:
- Display navigation menus
- Show secondary content
- Provide filtering options
- Present contextual tools
- Show user information

## Best Practices
- Keep navigation items organized and hierarchical
- Use clear, descriptive labels
- Include appropriate icons
- Support collapsible states
- Ensure responsive behavior
- Maintain consistent styling
- Consider mobile-first design
