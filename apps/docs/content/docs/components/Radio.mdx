# Usage Guidelines

Radio buttons allow users to select a single option from a set of choices. They can be used to:
- Select one option from a list
- Choose preferences or settings
- Make mutually exclusive selections
- Filter or sort content

## Best Practices
- Use clear, descriptive labels
- Group related options logically
- Provide clear visual feedback
- Ensure proper spacing between options
- Support keyboard navigation
- Consider mobile-friendly touch targets
- Use appropriate default selections
