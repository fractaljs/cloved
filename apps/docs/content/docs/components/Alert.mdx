# Usage Guidelines

Use alerts to communicate important information to users. They can be used to:
- Display success, error, warning, or informational messages
- Provide context for actions or system states
- Guide users through important processes

## Best Practices
- Keep alert messages clear and concise
- Use appropriate alert types based on the message severity
- Include actionable information when relevant
- Don't overuse alerts for non-critical information
