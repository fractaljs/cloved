# Usage Guidelines

TextArea allows users to input multiple lines of text. They can be used to:
- Write comments
- Enter descriptions
- Input long-form content
- Provide feedback
- Write messages

## Best Practices
- Set appropriate size limits
- Include character count when needed
- Provide clear placeholder text
- Support auto-resize when appropriate
- Include proper validation
- Consider accessibility
- Handle formatting options
