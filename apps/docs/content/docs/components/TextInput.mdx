# Usage Guidelines

TextInput allows users to enter single-line text. They can be used to:
- Enter usernames
- Input email addresses
- Search content
- Fill forms
- Enter short text

## Best Practices
- Use clear labels
- Include appropriate placeholder text
- Provide validation feedback
- Support keyboard navigation
- Consider input masks when needed
- Handle different input types
- Ensure proper error states
