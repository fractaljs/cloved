# Usage Guidelines

StatCard displays key metrics and statistics in a card format. They can be used to:
- Show important numbers
- Display performance metrics
- Present summary statistics
- Highlight key indicators
- Track progress

## Best Practices
- Keep information concise
- Use appropriate visual hierarchy
- Include relevant context
- Consider trend indicators
- Ensure readability
- Use consistent styling
- Consider responsive layout
