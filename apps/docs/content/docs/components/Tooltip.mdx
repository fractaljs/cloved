# Usage Guidelines

Tooltips provide additional information about elements when users hover or focus on them. They can be used to:
- Explain functionality
- Show additional details
- Provide context
- Display shortcuts
- Clarify actions

## Best Practices
- Keep content concise and helpful
- Position tooltips appropriately
- Use clear, simple language
- Ensure proper timing
- Support keyboard navigation
- Consider mobile interactions
- Avoid covering important content
